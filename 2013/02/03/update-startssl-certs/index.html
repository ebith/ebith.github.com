StartSSLの証明書類を更新する

Feb 3rd, 2013

メールアドレスとドメインの検証

Control PanelのAuthenticateからログイン。
Validations WizardでEmail Address ValidationとDomain Name Validationを行う。
両方30日で期限切れになるので証明書を更新するたびに必要。

クライアント認証書の更新

S/MIME and Authentication Certificateを選んで進めて行くと使用中のブラウザに証明書がインストールされるのでバックアップしておく。

サーバー証明書の更新

Web Server SSL/TLS Certificateを選んで秘密鍵を作り、サブドメインを決めて証明書を得る。

その他

サーバー証明書のところの秘密鍵は既にあるやつを持ってきても良いはずだけどなんだかうまく行かなかった。
